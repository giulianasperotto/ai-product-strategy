# The Prototype Bet

## What I Built
Two AI features demonstrated in a single prototype: an AI Copilot that helps instructors build conversion-optimized profiles faster (free, all instructors), and a Premium Scheduling Agent that automates booking management, reminders, and rescheduling for high-volume instructors ($19/month + reduced 8% commission).

## Tool Used
Lovable

## Prototype Link
https://waveupro.lovable.app

## AI Value Archetype
Copilot (Profile Generation) + Orchestrator (Scheduling Agent — chains actions: reads availability, drafts messages, confirms bookings, handles rescheduling)

## The Bet in One Sentence
If we help instructors build enriched profiles using AI and retain high-volume instructors through an AI scheduling agent that makes staying on WaveUp cheaper than transacting off-platform, then browse-to-booking conversion will increase from 8% to 15% and off-platform leakage will measurably decrease — because students will have enough proof of quality to trust a zero-review instructor, and instructors will have enough operational value to stay on the platform.

## Kill Criteria
Two kill criteria, one per feature:

Profile Generation Copilot: if browse-to-booking conversion for AI-generated profiles does not increase from 8% toward 15% within 3 months of launch, the profile enrichment hypothesis is invalidated — pivot to demand-side intervention (student matching AI).
